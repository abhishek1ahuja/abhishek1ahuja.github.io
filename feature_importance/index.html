Visualisation of Feature Dependencies

The Tree Diagram visualises feature importance of the dataset Breast Cancer The features are also orded in line with their importance from the subsets. The colour of the link represents whether the previous feature is included in the choice of subset and both, the width of the link and the radius of the node represent the importance of said feature. Scrolling over the node will highlight the path taken.

The Chord diagram visualises the feature dependencies of the same dataset. Here we can see which features share information by the links. The level of dependence is shown through the width of the links. By hovering over the links, one can see the calculated dependency score between the two.
